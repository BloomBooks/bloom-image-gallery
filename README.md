Currently, this is just the start of a project to re-implement the existing libpalaso Image Toolbox as react component. 

## Stack
* [ViteJS](https://vitejs.dev/)
* Typescript
* [MUI](https://mui.com/)
* [Emotion](https://emotion.sh)
